***# betterNemesis #*** 
---Shadow of war mod that expands on the nemesis system and adds new missions based on tribe. 

---#Project Features 
---This project will add many different features to the game. *(note theses aren't actual features of the mod currently these are being developed and might not be possible)*

-[+] tribal warfare: tribes will team together against other tribes. (teams can't raid each other) 

-[+] better Warchiefs: Warchiefs won't be completely fucking useless and can do missions when commanded. (basically SOM riots) 

-[+] better outposts: Outposts can reactivate and tribes will fight over them frequently.
