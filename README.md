# betterNemesis

A Shadow of War mod that expands the Nemesis System with dynamic world events, tribal conflicts, and improved AI behaviors. The goal is to make Mordor feel like a living world where Orc factions continue fighting for power even when the player is not directly involved.

> Note: This project is currently in development. Many planned features are experimental and may change or prove technically infeasible depending on engine limitations.

## Planned Features

### Tribal Warfare
- Tribes wage war against rival tribes across each region.
- Temporary tribal alliances can form against common enemies.
- Tribes compete for territory, influence, and outposts.
- Tribal raids occur dynamically instead of only through scripted DLC events.

### Improved Warchiefs
- Warchiefs actively command forces instead of remaining stationary.
- Warchiefs can initiate missions such as raids, duels, hunts, and assaults while still serving as Warchiefs.
- Inspired by the Riot mechanic from Shadow of Mordor.

### Dynamic Outposts
- Cleared outposts can eventually be reclaimed.
- Tribes launch assaults to capture enemy outposts.
- Outposts visually reflect the tribe currently controlling them.
- Capturing outposts affects regional tribal influence.

## Long-Term Goals

- Dynamic tribal influence system
- Living world simulation
- Tribe-specific soldier appearances and behaviors
- New Nemesis missions
- Better captain interactions
- Improved fortress activity
- Dynamic patrols and tribal encounters
